## SCM 产品名称下拉选中后闪烁修复
- 用户反馈新增产品界面选择金蝶物料后，产品名称输入框内容重复闪烁
- 根因：
  - `value` 从 `kdMaterials.find(...)` 动态计算，options 变化时对象引用改变
  - `onInputChange` 在选择/重置时也会触发搜索，导致 kdMaterials 更新 → value 重算 → 输入框闪烁
- 修复：
  - 增加 `kdInput` 和 `kdSelected` 两个独立 state
  - `value` 使用稳定的 `kdSelected` 对象，不再从 options 中动态查找
  - `onInputChange` 仅当 `reason === 'input'`（用户输入）时才触发搜索
  - 选择时更新 `kdSelected` 和 `kdInput`，输入框保持稳定
- 重新构建前端 + 重启 gateway
