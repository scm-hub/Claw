### 移动端 API 响应解析错误修复
- **根因**：多个页面 `PurchaseOrders.jsx`、`Payables.jsx`、`Receivables.jsx` 错误使用 `res.data.data.list`
- `api.get()` 返回的是已解析的 JSON 对象（`{ success, data: { list, total } }`），不是 axios 的 response wrapper
- 正确用法：`res.data.list` / `res.data.total`（去掉一层 `.data`）
